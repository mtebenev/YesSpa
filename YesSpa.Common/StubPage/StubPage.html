YesSpa Stub Page

YesSpa serves this URL in favor of embedded SPA.

Please use native environment in development mode (@angular/cli or create-react-app) to develop the SPA.
Important: please enable CORS to make API calls.

In production environment this page will be replaced with actual SPA.

To render actual embedded application in development mode use UseStubPage flag in Startup.Configure(): 

      
        app.UseSpa(builder =>
        {
          builder.Options.UseStubPage = false;
          builder.AddSpa(...);
          ...
        });
